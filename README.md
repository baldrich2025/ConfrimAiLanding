# ConfirmAI Landing Page

The official landing page for ConfirmAI - an AI appointment call automation system for small businesses.

## Overview

ConfirmAI automates outbound and inbound appointment calls using AI voice agents. This repository contains the landing page code for the ConfirmAI product.

## Tech Stack

- React + Next.js
- Tailwind CSS
- Deployed via GitHub Pages

## Main Features Showcased

- AI-powered appointment confirmation calls
- Reduce missed appointments by up to 80%
- Integration with existing scheduling systems
- Natural-sounding voice technology

## Development

### Prerequisites

- Node.js 18+ and npm

### Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```bash
# Create production build
npm run build

# Preview production build locally
npx serve out
```

## Deployment

This site is automatically deployed via GitHub Actions when changes are pushed to the main branch.
